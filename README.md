# BenchmarkApp

The project displays general information about PC hardware. Then, to test the performance, the project runs some tests to generate relevant information. To grasp the performance of a PC each test should run multiple times and make an average out of these test runs to arrive at predictable, useful conclusions.

IMPORTANT: <br>
Inside the .py the PATH must be changed to the .exe generated after running the C files. <br>
The .cpp files must be inside the same folder when running or paths must be changed.
